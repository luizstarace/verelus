# Verelus

Marca guarda-chuva para produtos SaaS B2B no Brasil.

## Atalaia (primeiro produto)

Atendente IA 24/7 para PMEs brasileiras via WhatsApp + widget de chat. Para clínicas, restaurantes, salões, lojas e serviços que recebem mensagens fora do horário e não querem perder cliente.

- **URL**: [atalaia.verelus.com](https://atalaia.verelus.com)
- **Stack**: Next.js 14, Supabase, Stripe, Claude Haiku 4.5, ElevenLabs, Evolution API (WhatsApp self-hosted), Cloudflare Pages, Cloudflare Tunnel
- **Modelo**: SaaS híbrido — Starter R$147 / Pro R$297 / Business R$597 + excedente por uso. Trial 7 dias sem cartão.
- **Docs**: [`Atalaia/docs/spec.md`](./Atalaia/docs/spec.md)
